                             TRABALHO DE CONCLUSÃO MASTER GEEKS 2:
-----------------------------------------------------------------------------------------------------------

Seu objetivo é projetar e desenvolver um site utilizando as linguagens HTML e CSS aprendidas em aula.
A escolha da temática do site é livre por parte do aluno (por exemplo: um portifólio, fórum, loja online,
jogo, landing page, portal de notícias, streaming, entre outros), o site pode ser sobre algum assunto
que você gosta!

Você é livre para se inspirar em sites já existentes e utilizar como base elementos populares como por
exemplo navbars, cards e containers. Mas procure dar uma identidade visual para seu projeto, criando um
logo, pense na paleta de cores, fonte e estilo dos elementos que irá utilizar, use a criatividade ao seu favor!

Lembre-se que este será um site com propósito de aprendizado, no final ele poderá ser publicado na web
pelo Github para ser acessado mas as interações, formulários, botões, compras não irão "funcionar de
verdade" porque as páginas por enquanto são estáticas. Aprenderemos a torna-las dinâmicas no Master
Geeks 3 com a linguagem Javascript!

Observações: Atente-se à manter os códigos e arquivos organizados, procure salvar no Github no fim de
toda aula para não perder seu progresso. Consulte seu instrutor sempre que precisar de ajuda. No final do
curso, os projetos serão apresentados pelos alunos para os pais e poderá ser uma oportunidade de empresas
começarem a conhecer o trabalho de vocês!
